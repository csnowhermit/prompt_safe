2. 安全网关模块详细设计

2.1 功能描述

安全网关是整个系统的入口和流程编排中心，负责接收用户请求、依次调用脱敏和安全检查模块、转发请求至大模型、处理大模型响应并返回用户。

2.2 核心处理流程

3.2.2 规则优先级

为避免短模式错误匹配长模式中的子串，规则按以下优先级执行：

优先级从高到低: 1. R01 身份证号(18位) — 最长最精确，优先匹配 2. R02 身份证号(15位) — 长且含日期结构 3. R04 银行卡号 — 需Luhn校验，16-19位 4. R09 车牌号 — 含中文字符，特征明显 5. R10 统一社会信用代码 — 18位字母数字混合 6. R14 军官证号 — 含"军字第"特征 7. R16/R17 港澳台通行证 — 字母+8位数字 8. R08 护照号 — 字母+8位数字 9. R03 手机号 — 1[3-9]开头11位 10. R05 邮箱 — @符号特征明显 11. R06/R07 IP地址 — 点分隔特征 12. R15 社保卡号 — 纯数字15/18位 13. R13 IMEI号 — 15位纯数字 14. R11 QQ号 — 5-11位数字 15. R12 微信号 — 字母开头6-20位（误报率高，放最后） 16. R18-R20 其他证照号 — 结合上下文关键词判断
误报控制策略：对于纯数字类规则（QQ号、IMEI、银行卡号等），通过以下方式降低误报率：
  • 银行卡号使用 Luhn算法 校验，过滤非真实卡号的长数字串
  • 身份证号使用 地区码+出生日期+校验位 三重验证
  • QQ号要求不在已有更长匹配（如手机号、身份证号）的范围内
  • 微信号要求不以常见英文单词开头（维护白名单）
  • 营业执照号/纳税识别号要求上下文含"营业执照""纳税"等关键词

3.3 脱敏引擎核心算法

class MaskEngine: """敏感信息脱敏引擎""" def __init__(self, rules: list[MaskRule]): # 按优先级排序规则 self.rules = sorted(rules, key=lambda r: r.priority) # 预编译正则 self.compiled = [(r.id, re.compile(r.pattern), r) for r in self.rules] async def mask(self, text: str) -> MaskResult: """ 对文本执行脱敏处理 返回: MaskResult(masked_text, masked_count, details) """ if not text or not text.strip(): return MaskResult(text, 0, []) masked_text = text details = [] masked_count = 0 # 已被匹配的字符位置区间（避免重复匹配） occupied = set() for rule_id, regex, rule in self.compiled: for match in regex.finditer(masked_text): start, end = match.start(), match.end() # 检查是否已被更高优先级的规则匹配 if any(i in occupied for i in range(start, end)): continue original = match.group() # 执行校验（如Luhn校验、身份证校验位校验） if rule.validator and not rule.validator(original): continue # 执行掩码替换 masked_value = rule.mask_func(original) details.append(MaskDetail( rule_id=rule_id, type=rule.type_name, position=(start, end), original=original, # 仅内存中保留，不写入日志 masked=masked_value )) # 标记区间已被占用 occupied.update(range(start, end)) masked_count += 1 # 从后往前替换（避免位置偏移） for detail in sorted(details, key=lambda d: d.position[0], reverse=True): start, end = detail.position masked_text = masked_text[:start] + detail.masked + masked_text[end:] return MaskResult( masked_text=masked_text, masked_count=masked_count, details=details ) def luhn_check(card_number: str) -> bool: """Luhn算法校验银行卡号""" digits = [int(d) for d in card_number if d.isdigit()] if len(digits) < 16: return False checksum = 0 for i, d in enumerate(reversed(digits)): if i % 2 == 1: d *= 2 if d > 9: d -= 9 checksum += d return checksum % 10 == 0 def id_card_check(id_number: str) -> bool: """身份证号校验位验证""" if len(id_number) != 18: return False # 校验位权重 weights = [7,9,10,5,8,4,2,1,6,3,7,9,10,5,8,4,2] check_codes = "10X98765432" total = sum(int(id_number[i]) * weights[i] for i in range(17)) return id_number[17].upper() == check_codes[total % 11]

4. 语义安全检查模块详细设计

4.1 功能描述

安全检查引擎（SafetyChecker）负责调用 ynygljj/Unified_Prompt_Guard 模型，对文本进行安全/不安全分类判定。

4.2 模型推理服务设计

4.2.1 模型部署

4.2.2 推理接口设计

class SafetyChecker: """语义安全检查引擎""" def __init__(self, model_endpoint: str, threshold: float = 0.5, timeout: float = 3.0): self.endpoint = model_endpoint # vLLM服务地址 self.threshold = threshold # 安全判定阈值 self.timeout = timeout # 超时时间(秒) async def check(self, text: str) -> SafetyCheckResult: """ 对文本执行安全检查 返回: SafetyCheckResult(is_safe, confidence, label, error) """ if not text or not text.strip(): # 空文本默认安全 return SafetyCheckResult(True, 0.0, "safe", None) try: # 调用模型推理服务 result = await self._call_model(text) label = result["label"] # "safe" or "unsafe" score = result["score"] # 0.0 ~ 1.0 if label == "safe" and score >= self.threshold: return SafetyCheckResult(True, score, "safe", None) elif label == "unsafe" and score >= self.threshold: return SafetyCheckResult(False, score, "unsafe", None) else: # 模型不确定（score < threshold），保守拦截 return SafetyCheckResult(False, score, "uncertain", "模型置信度低于阈值") except asyncio.TimeoutError: # 超时 → Fail-Closed return SafetyCheckResult(False, 0.0, "timeout", "安全检查超时，按安全策略拦截") except ModelUnavailableError: # 模型不可用 → 降级处理 return SafetyCheckResult(False, 0.0, "unavailable", "安全检查服务不可用") async def _call_model(self, text: str) -> dict: """调用vLLM推理服务""" async with httpx.AsyncClient(timeout=self.timeout) as client: resp = await client.post( f"{self.endpoint}/v1/classify", json={"text": text} ) resp.raise_for_status() return resp.json()

4.2.3 判定决策矩阵

模型输出置信度判定结果处理动作
safe≥ 阈值(0.5)安全PASS 放行
safe< 阈值不确定BLOCK 保守拦截
unsafe≥ 阈值(0.5)不安全BLOCK 拦截
unsafe< 阈值不确定BLOCK 保守拦截
超时-检查失败BLOCK Fail-Closed

4.3 降级策略

降级模式触发条件行为适用场景
Fail-Closed（默认）模型不可用/超时拦截所有请求，返回安全提示高安全要求场景
Fail-Open模型不可用/超时仅执行脱敏后放行，跳过语义检查低风险场景/灰度期
Only-Mask手动配置仅脱敏不检查，永久跳过语义检查调试/测试场景

4.4 批量推理优化

class BatchSafetyChecker(SafetyChecker): """支持批量推理的安全检查器""" def __init__(self, *args, batch_size=32, **kwargs): super().__init__(*args, **kwargs) self.batch_size = batch_size async def check_batch(self, texts: list[str]) -> list[SafetyCheckResult]: """批量检查多段文本""" results = [] for i in range(0, len(texts), self.batch_size): batch = texts[i:i+self.batch_size] batch_results = await self._call_model_batch(batch) results.extend(batch_results) return results async def _call_model_batch(self, texts: list[str]) -> list[SafetyCheckResult]: """调用vLLM批量推理接口""" async with httpx.AsyncClient(timeout=self.timeout) as client: resp = await client.post( f"{self.endpoint}/v1/classify/batch", json={"texts": texts} ) data = resp.json() return [ SafetyCheckResult( is_safe=(r["label"] == "safe" and r["score"] >= self.threshold), confidence=r["score"], label=r["label"], error=None ) for r in data["results"] ]

7.2 配置热更新

脱敏规则支持热更新，无需重启服务：

# 脱敏规则热更新流程: 1. 修改 config/mask_rules.yaml 2. POST /api/v1/admin/rules/reload 3. 网关重新加载规则文件，预编译正则 4. 新规则立即生效（对后续请求生效） 5. 返回加载结果（成功/失败 + 规则数量） # 安全阈值热更新: POST /api/v1/admin/config { "safety.threshold": 0.6, "safety.degradation": "fail_open" }
